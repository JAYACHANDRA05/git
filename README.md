# Educational E-Commerce Project

## ⚠️ Disclaimer
This is an **educational learning project** created for practicing HTML and CSS web development. This project is **NOT affiliated with Amazon or any real e-commerce platform**.

## Project Overview
This is a frontend learning project demonstrating how to build a responsive e-commerce website interface using HTML and CSS. The design is inspired by modern e-commerce platforms for educational purposes only.

## Language Composition
- HTML: 59.8%
- CSS: 40.2%

## Features
- Responsive navigation bar
- Hero section with promotional messaging
- Product category showcase
- Footer with links and information
- Font Awesome icons for UI elements
- Clean and modern design
